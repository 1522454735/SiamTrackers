# Pytorch implementation of DaSiamRPN
## Description

Code:https://github.com/HonglinChu/DaSiamRPN

压缩包包含完整的程序，可测试和评估,支持pysot toolkit ; 将python2版本修改为python3版本,另外需要你下载SiamRPNBIG模型; 在vot2016和vot2018上可以复现出和论文一致的结果
